# 🚀 AI Marketing Campaign Generator

An AI-powered tool to generate complete marketing campaigns (Email Copy, Social Ads, Blog, Landing Page) and automatically save **email drafts** to Gmail.

## Features

- Uses **LangChain + Groq/OpenAI** for content generation
- Saves campaign email copy to **Gmail Drafts** via API
- Streamlit-powered UI
- Docker + GitHub Actions ready

## Run locally

```bash
pip install -r requirements.txt
streamlit run app/main.py
```
